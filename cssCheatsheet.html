🎨 Comprehensive CSS Cheatsheet

Your complete reference guide to CSS selectors, properties, and concepts

🧱 Foundation

📜 CSS Basics

Concept Description Example
CSS Rule A CSS rule consists of a selector and a declaration block. p { color: red; }
Selector The selector points to the HTML element you want to style. h1, .class, #id
Declaration The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. color: red; font-size: 16px;
Comments Used to explain the code, and may help when you edit the source code at a later date. Comments are ignored by browsers. /* This is a CSS comment */

🎯 Selectors

Basic Selectors

Selector Description Example
* Universal selector, selects all elements. * { margin: 0; }
element Type selector, selects all elements of a given type. p { line-height: 1.5; }
.class Class selector, selects all elements with a specific class attribute. .highlight { background-color: yellow; }
#id ID selector, selects a single element with a specific id attribute. #header { font-size: 24px; }
[attribute] Attribute selector, selects elements with a specific attribute. [required] { border-color: red; }
[attribute=value] Attribute selector, selects elements with a specific attribute and value. [type="submit"] { background-color: blue; }

Grouping and Combinators

Selector Description Example
A, B Grouping selector, selects all A and B elements. h1, h2 { color: navy; }
A B Descendant selector, selects all B elements inside an A element. div p { color: gray; }
A > B Child selector, selects all B elements that are a direct child of an A element. ul > li { list-style-type: none; }
A + B Adjacent sibling selector, selects the B element that is placed immediately after an A element. h1 + p { margin-top: 0; }
A ~ B General sibling selector, selects all B elements that are siblings of an A element. h1 ~ p { color: green; }

Pseudo-classes

Selector Description Example
:hover Selects an element when the user hovers over it. a:hover { text-decoration: underline; }
:focus Selects an element that has focus (e.g., an input field). input:focus { border-color: blue; }
:first-child Selects the first child of its parent. li:first-child { font-weight: bold; }
:last-child Selects the last child of its parent. li:last-child { border-bottom: none; }
:nth-child(n) Selects elements based on their order within a parent. Can be a number, keyword (even, odd), or formula. tr:nth-child(even) { background-color: #f2f2f2; }
:not(selector) Selects every element that is NOT the specified selector. p:not(.special) { color: black; }

Pseudo-elements

Selector Description Example
::before Inserts content before the content of an element. p::before { content: "Note: "; }
::after Inserts content after the content of an element. a::after { content: " »"; }
::first-letter Selects the first letter of an element. p::first-letter { font-size: 2em; }
::first-line Selects the first line of an element. p::first-line { font-weight: bold; }

🖋️ Text & Font Properties

Property Description Example Values
color Sets the color of the text. red, #ff0000, rgb(255,0,0)
font-family Specifies a prioritized list of one or more font family names. "Arial", sans-serif, "Times New Roman", serif
font-size Sets the size of the font. 16px, 1.2em, 120%, large
font-weight Sets the thickness of the font. normal, bold, 400, 700
font-style Sets the style of the font (e.g., normal, italic). normal, italic, oblique
text-align Sets the horizontal alignment of text. left, right, center, justify
text-decoration Adds decoration to text. none, underline, overline, line-through
line-height Sets the distance between lines of text. normal, 1.6, 20px, 150%
letter-spacing Controls the space between characters. normal, 2px, -1px
text-transform Controls the capitalization of text. none, capitalize, uppercase, lowercase
text-indent Specifies the indentation of the first line in a text-block. 50px, 2em, -20px
text-shadow Adds shadow to text. 2px 2px 5px black, 0 0 3px rgba(0,0,0,0.5)
vertical-align Sets the vertical alignment of an element. baseline, middle, top, bottom
white-space Specifies how white-space inside an element is handled. normal, nowrap, pre, pre-wrap, pre-line
text-overflow Specifies how overflowed content that is not displayed should be signaled to the user. clip, ellipsis, "..."
word-spacing Increases or decreases the white space between words. normal, 10px, -2px

📦 Box Model

💡 Tip: The box model is a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
Property Description Example Values
width Sets the width of an element. 100px, 50%, auto
height Sets the height of an element. 100px, 50vh, auto
padding Sets the space between the content and the border. Shorthand for top, right, bottom, left. 10px, 10px 20px, 10px 20px 30px 40px
margin Sets the space outside the border, clearing an area around the element. Shorthand for top, right, bottom, left. 10px, 10px 20px, 10px 20px 30px 40px, auto
border Shorthand property to set the width, style, and color of an element's border. 1px solid black
border-radius Rounds the corners of an element's outer border edge. 5px, 50%
box-sizing Changes how the total width and height of an element is calculated. content-box, border-box
box-shadow Adds shadow effects around an element's frame. 2px 2px 5px black, inset 0 0 10px rgba(0,0,0,0.1)
outline Adds an outline around elements (outside the border). 1px solid blue, none
outline-offset Sets the space between an outline and the border edge of an element. 2px, -5px
min-width Sets the minimum width of an element. 100px, 50%, auto
max-width Sets the maximum width of an element. 100px, 50%, none
min-height Sets the minimum height of an element. 100px, 50vh, auto
max-height Sets the maximum height of an element. 100px, 50vh, none

🎨 Background & Colors

Property Description Example Values
background-color Sets the background color of an element. blue, #0000ff, rgba(0,0,255,0.5)
background-image Sets one or more background images for an element. url('image.jpg')
background-size Specifies the size of the background images. auto, cover, contain, 50%, 100px 100px
background-position Specifies the initial position of a background image. center, top left, 50% 50%
background-repeat Sets if/how a background image will be repeated. no-repeat, repeat-x, repeat-y, repeat
opacity Sets the opacity level for an element. 0.5, 1, 0

📐 Layout & Positioning

Property Description Example Values
display Specifies the display behavior of an element. block, inline, inline-block, flex, grid, none
position Specifies the type of positioning method used for an element. static, relative, absolute, fixed, sticky
top / right / bottom / left Specifies the final position of a positioned element. 10px, -2em, auto
z-index Specifies the stack order of a positioned element. auto, 1, 999, -1
float Specifies whether an element should float to the left, right, or not at all. left, right, none
clear Specifies what elements can float beside the cleared element and on which side. left, right, both, none

Flexbox

Property Description (on container) Example Values
display: flex; Defines a flex container. —
flex-direction Establishes the main-axis, thus defining the direction flex items are placed in the flex container. row, row-reverse, column, column-reverse
justify-content Aligns flex items along the main axis. flex-start, flex-end, center, space-between, space-around
align-items Aligns flex items along the cross axis. stretch, flex-start, flex-end, center, baseline
flex-wrap Allows flex items to wrap onto multiple lines. nowrap, wrap, wrap-reverse
flex (on items) A shorthand for flex-grow, flex-shrink, and flex-basis. 1, 0 1 auto, 2 0 200px
flex-grow (on items) Specifies how much a flex item will grow relative to the rest. 0, 1, 2
flex-shrink (on items) Specifies how much a flex item will shrink relative to the rest. 1, 0
flex-basis (on items) Specifies the initial length of a flex item. 200px, auto
order (on items) Specifies the order used to lay out a flex item in its flex container. 1, -1, 0
align-self (on items) Allows the default alignment (or the one specified by align-items) to be overridden. stretch, center, flex-start, flex-end

Grid

Property Description (on container) Example Values
display: grid; Defines a grid container. —
grid-template-columns Defines the columns of the grid. 100px 100px, 1fr 2fr, repeat(3, 1fr)
grid-template-rows Defines the rows of the grid. 100px auto, repeat(2, 100px)
gap A shorthand for row-gap and column-gap. 10px, 10px 20px
grid-column (on items) A shorthand for grid-column-start and grid-column-end. 1 / 3, span 2, auto
grid-column-start (on items) Specifies where to start displaying the item within the grid column. 1, span 2
grid-column-end (on items) Specifies where to end displaying the item within the grid column. 3, span 2
grid-row (on items) A shorthand for grid-row-start and grid-row-end. 1 / 3, span 1
grid-row-start (on items) Specifies where to start displaying the item within the grid row. 1, span 1
grid-row-end (on items) Specifies where to end displaying the item within the grid row. 2, span 1
grid-area (on items) Either specifies a grid item's size and location within a grid layout, or establishes a named grid area. 1 / 2 / 3 / 4, header
justify-self (on items) Aligns a grid item inside its grid area along the inline (row) axis. start, end, center, stretch
align-self (on items) Aligns a grid item inside its grid area along the block (column) axis. start, end, center, stretch

🎬 Transitions & Animations

Property Description Example Values
transition A shorthand property for transitioning one or more properties between two states. all 0.3s ease, width 2s linear
transition-property Specifies the name of the CSS property to apply the transition effect to. width, background-color, all
transition-duration Specifies how many seconds or milliseconds a transition effect takes to complete. 0.5s, 200ms
transition-timing-function Specifies the speed curve of the transition effect. ease, linear, ease-in-out, cubic-bezier(0.1,0.7,1,0.1)
transition-delay Specifies when the transition effect will start. 0.5s, 100ms
animation A shorthand property for animating one or more properties with keyframes. slidein 3s ease-in 1s infinite reverse both running
@keyframes Defines the animation keyframes. @keyframes fade { from {opacity: 0;} to {opacity: 1;} }

🔄 Transforms

Property Description Example Values
transform Allows 2D or 3D transformation to be applied to an element. translate(50px, 100px), rotate(45deg), scale(1.5, 1.5)
transform-origin Sets the origin for transformations performed on an element. center, 50% 50%, top left
perspective Defines the viewing distance of the user from the z-plane origin. 300px, none
perspective-origin Defines where the user is looking at (the origin) on the z-plane. 50% 50%, top center
backface-visibility Sets whether the back face of an element is visible when turned towards the user. visible, hidden

📱 Responsive Design

💡 Tip: Media queries allow you to apply different styles for different devices/screens. Common breakpoints: mobile (< 768px), tablet (768px - 1024px), desktop (> 1024px).
Property/Syntax Description Example
@media (max-width: 768px) { ... } Applies styles only if the viewport width is less than or equal to 768px. @media (max-width: 768px) { body { font-size: 14px; } }
@media (min-width: 769px) Applies styles only if the viewport width is greater than or equal to 769px. @media (min-width: 769px) { body { font-size: 16px; } }
@media screen and (orientation: landscape) Applies styles based on screen orientation. @media screen and (orientation: landscape) { div { width: 50%; } }
@media print { ... } Applies styles when the document is printed. @media print { body { color: black; } }
