Adding Images

The tag embeds images. It's self-closing and requires at least the src attribute.



Important Image Attributes

  • src - Image file path or URL
  • alt - Alternative text (required for accessibility)
  • width/height - Size in pixels
  • loading="lazy" - Load image only when needed
Accessibility Rule: Always include meaningful alt text that describes the image for screen readers and when images fail to load.

Example: For a photo of a cat, use alt="Orange tabby cat sleeping on windowsill" instead of just alt="Cat".

Image Formats

Common web image formats:

  • JPEG/JPG - Best for photos (lossy compression)
  • PNG - Best for graphics with transparency
  • GIF - Animated images (limited colors)
  • WebP - Modern format with better compression
  • SVG - Vector graphics (scalable, no pixelation)

Best Practices

  • Optimize images for web (reduce file size without losing quality)
  • Use descriptive filenames (logo.png instead of img123.png)
  • Always include alt text for accessibility
  • Use relative paths for images in your project
  • Consider image lazy loading for better performance
  • Test all links to ensure they work
