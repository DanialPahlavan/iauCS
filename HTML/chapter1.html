Chapter 1: Getting Started with HTML

Learn the basics of HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML describes the structure of a webpage and tells the browser how to display content.

HTML is not a programming language - it's a markup language that uses tags to define elements like headings, paragraphs, links, images, and more.

Did you know? HTML was invented by Tim Berners-Lee in 1989 and has evolved through several versions. The current standard is HTML5.
