۱. HTML چیست؟

HTML زبان استاندارد برای ساخت صفحات وب و اسکلت اصلی هر وب‌سایتی است. این زبان به مرورگر می‌گوید که محتوا را چگونه نمایش دهد. این کار با استفاده از "تگ‌ها" (Tags) انجام می‌شود.

ساده‌ترین سند HTML

<!DOCTYPE html>
<html>
<head>
  <title>اولین صفحه من</title>
</head>
<body>
  <h1>سلام دنیا!</h1>
  <p>اولین پاراگراف من در وب.</p>
</body>
</html>

تگ‌های اصلی و کاربردی

<!-- تیترها از h1 تا h6 -->
<h1>تیتر اصلی</h1>
<h2>تیتر سطح دوم</h2>
<p>پاراگراف معمولی از متن.</p>

<!-- لینک‌ها -->
<a href="https://example.com">بازدید وب‌سایت</a>

<!-- تصویر -->
<img src="image.jpg" alt="توضیح تصویر">

<!-- لیست‌ها -->
<ul>
  <li>آیتم اول</li>
  <li>آیتم دوم</li>
</ul>

۲. ساختار اصلی یک سند HTML

هر فایل HTML یک ساختار استاندارد دارد که به مرورگر در درک محتوا کمک می‌کند.

<!DOCTYPE html>
<html lang="fa" dir="rtl">
  <head>
    <!-- فراداده و اطلاعات صفحه اینجا قرار می‌گیرد -->
  </head>
  <body>
    <!-- محتوای قابل مشاهده اینجا قرار می‌گیرد -->
  </body>
</html>

۳. بخش <head> و فراداده (Metadata)

بخش <head> شامل اطلاعاتی درباره صفحه است که مستقیماً به کاربر نمایش داده نمی‌شود، اما برای مرورگرها و موتورهای جستجو بسیار مهم است.

<head>
  <!-- انکدینگ کاراکتر برای پشتیبانی از زبان فارسی -->
  <meta charset="UTF-8">
  
  <!-- برای طراحی واکنش‌گرا (Responsive Design) -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- توضیحات صفحه برای موتورهای جستجو (SEO) -->
  <meta name="description" content="راهنمای جامع آموزش HTML برای مبتدیان.">
  
  <!-- عنوان صفحه که در تب مرورگر نمایش داده می‌شود -->
  <title>عنوان صفحه</title>
  
  <!-- اتصال فایل CSS خارجی -->
  <link rel="stylesheet" href="styles.css">
</head>

۴. کار با محتوای اصلی

تیترها و پاراگراف‌ها

از تگ‌های <h1> تا <h6> برای تیترها و از <p> برای پاراگراف‌ها استفاده می‌شود.

<h1>این بالاترین سطح تیتر است</h1>
<h2>تیتر سطح دوم</h2>
<h3>تیتر سطح سوم</h3>
<p>این یک پاراگراف ساده است.</p>
<p>پاراگراف دوم با <br> شکست خط.</p>

لینک‌ها و تصاویر

تگ <a> برای ایجاد لینک و تگ <img> برای نمایش تصاویر است.

<!-- لینک خارجی -->
<a href="https://google.com">بازدید گوگل</a>

<!-- لینک داخلی -->
<a href="#section1">برو به بخش اول</a>

<!-- تصویر با متن جایگزین -->
<img src="photo.jpg" alt="توضیح تصویر" width="300" height="200">

<!-- تصویر با لینک -->
<a href="gallery.html">
  <img src="thumbnail.jpg" alt="گالری تصاویر">
</a>

لیست‌ها

برای ایجاد لیست‌های نامرتب (<ul>) و مرتب (<ol>) استفاده کنید.

<!-- لیست نامرتب -->
<ul>
  <li>سیب</li>
  <li>پرتقال</li>
  <li>موز</li>
</ul>

<!-- لیست مرتب -->
<ol>
  <li>صبحانه بخور</li>
  <li>دندان‌ها را مسواک بزن</li>
  <li>به مدرسه برو</li>
</ol>

<!-- لیست تو در تو -->
<ul>
  <li>میوه‌ها
    <ul>
      <li>سیب</li>
      <li>پرتقال</li>
    </ul>
  </li>
  <li>سبزیجات</li>
</ul>

قالب‌بندی پیشرفته متن

این متن مهم است و این متن تأکید دارد.

H2O و E = mc2

این یک نقل قول است.
<!-- متن ضخیم و کج -->
<p>متن <strong>ضخیم</strong> و <em>کج</em>.</p>

<!-- زیرنویس و بالانویس -->
<p>فرمول آب: H<sub>2</sub>O</p>
<p>معادله: E = mc<sup>2</sup></p>

<!-- نقل قول -->
<blockquote>
  برنامه‌نویسی هنر تبدیل کافئین به کد است.
</blockquote>

<!-- خط جداکننده -->
<hr>

<!-- متن از پیش قالب‌بندی شده -->
<pre>
این متن
  قالب‌بندی
    خود را حفظ می‌کند
</pre>

۵. ویژگی‌ها (Attributes) در HTML

ویژگی‌ها اطلاعات بیشتری درباره عناصر HTML ارائه می‌دهند و به صورت زوج‌های نام="مقدار" در تگ باز قرار می‌گیرند.

  • id: یک شناسه منحصر به فرد.
  • class: برای گروه‌بندی و استایل‌دهی چند عنصر.
  • style: برای استایل‌دهی مستقیم (inline).
  • title: برای نمایش توضیحات تکمیلی (tooltip).
  • disabled: یک ویژگی بولی (boolean) برای غیرفعال کردن یک عنصر ورودی.
<p class="highlight" id="intro-p">یک پاراگراف با کلاس و آیدی.</p>
<button title="این دکمه غیرفعال است" disabled>کلیک نکنید</button>

۷. رسانه و گرافیک

صدا و ویدیو

با تگ‌های <audio> و <video> می‌توانید فایل‌های چندرسانه‌ای را در صفحه خود پخش کنید.

<!-- پخش صدا -->
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
  مرورگر شما از تگ audio پشتیبانی نمی‌کند.
</audio>

<!-- پخش ویدیو -->
<video width="640" height="360" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.webm" type="video/webm">
  مرورگر شما از تگ video پشتیبانی نمی‌کند.
</video>

گرافیک وکتوری (SVG)

SVG برای نمایش آیکون‌ها و گرافیک‌های مقیاس‌پذیر بدون افت کیفیت عالی است.

<!-- آیکون SVG ساده -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">
  <path d="M12 2L15.09 8.26L22 9L17 14L18.18 21L12 17.77L5.82 21L7 14L2 9L8.91 8.26L12 2Z" fill="currentColor"/>
</svg>

<!-- دایره SVG -->
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="#4f46e5" stroke-width="4" fill="#e5e7eb" />
</svg>

گرافیک پیکسلی (Canvas)

تگ <canvas> یک بوم نقاشی برای شما فراهم می‌کند که می‌توانید با جاوااسکریپت روی آن طراحی کنید. این تگ برای بازی‌ها و گرافیک‌های پیچیده کاربرد دارد.

<!-- عنصر Canvas -->
<canvas id="myCanvas" width="400" height="200" style="border:1px solid #000;">
  مرورگر شما از Canvas پشتیبانی نمی‌کند.
</canvas>

<!-- JavaScript برای رسم روی Canvas (در فایل جداگانه) -->
<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');

  // رسم مستطیل
  ctx.fillStyle = '#4f46e5';
  ctx.fillRect(10, 10, 150, 100);

  // رسم متن
  ctx.font = '20px Arial';
  ctx.fillText('سلام Canvas!', 10, 140);
</script>

۱۰. دسترسی‌پذیری وب (Accessibility)

نوشتن HTML دسترسی‌پذیر به این معناست که وب‌سایت شما برای همه افراد، از جمله افراد دارای معلولیت، قابل استفاده باشد. این یک بخش حیاتی از توسعه وب مدرن است.

  • استفاده از HTML معنایی: ساختار درست به صفحه‌خوان‌ها کمک می‌کند.
  • ویژگی alt برای تصاویر: همیشه متنی جایگزین برای تصاویر قرار دهید.
  • برچسب برای فرم‌ها: هر ورودی در فرم باید یک برچسب (<label>) مرتبط داشته باشد.
  • استفاده از ویژگی‌های ARIA: در موارد پیچیده، از ویژگی‌های ARIA برای توصیف نقش و وضعیت عناصر استفاده کنید.
<!-- تصویر با متن جایگزین مناسب -->
<img src="logo.png" alt="لوگوی شرکت ما">

<!-- دکمه‌ای که فقط آیکون دارد و با ARIA توصیف شده -->
<button aria-label="بستن پنجره">
  <!-- SVG icon -->
  <svg>...</svg>
</button>

۱۱. کامنت‌ها و نکات مهم

کامنت‌گذاری در HTML

کامنت‌ها توسط مرورگر نمایش داده نمی‌شوند اما به شما و دیگران در درک کد کمک می‌کنند.

<!-- این یک کامنت تک خطی است -->

<!--
  این یک کامنت
  چند خطی است
  که توضیحات بیشتری ارائه می‌دهد
-->

نکات مهم کدنویسی (Best Practices)

  • همیشه تگ‌ها را ببندید: این کار از خطاهای غیرمنتظره جلوگیری می‌کند.
  • از حروف کوچک استفاده کنید: نام تگ‌ها و ویژگی‌ها را با حروف کوچک بنویسید.
  • کد خود را مرتب کنید: از تورفتگی (Indentation) برای خوانایی بهتر استفاده کنید.
  • از HTML معنایی استفاده کنید: برای بهبود سئو و دسترسی‌پذیری.
  • ویژگی‌های مناسب را اضافه کنید: مانند alt برای تصاویر و title برای لینک‌ها.
<!-- اشتباه: تگ بسته نشده -->
<p>این پاراگراف بسته نشده است

<!-- درست: تگ بسته شده -->
<p>این پاراگراف درست بسته شده است.</p>

<!-- اشتباه: حروف بزرگ -->
<DIV>این اشتباه است</DIV>

<!-- درست: حروف کوچک -->
<div>این درست است</div>

۱۳. فرم‌های پیشرفته

فرم‌های HTML انواع مختلفی از ورودی‌ها را پشتیبانی می‌کنند.

انواع ورودی‌های مختلف

<!-- ورودی متن -->
<label for="username">نام کاربری:</label>
<input type="text" id="username" name="username" required>

<!-- ورودی ایمیل -->
<label for="email">ایمیل:</label>
<input type="email" id="email" name="email" required>

<!-- ورودی گذرواژه -->
<label for="password">گذرواژه:</label>
<input type="password" id="password" name="password" required>

<!-- ورودی شماره -->
<label for="age">سن:</label>
<input type="number" id="age" name="age" min="1" max="120">

<!-- ورودی تاریخ -->
<label for="birthdate">تاریخ تولد:</label>
<input type="date" id="birthdate" name="birthdate">

<!-- ورودی زمان -->
<label for="meeting-time">زمان ملاقات:</label>
<input type="datetime-local" id="meeting-time" name="meeting-time">

<!-- انتخاب چند گزینه‌ای -->
<label for="country">کشور:</label>
<select id="country" name="country">
  <option value="">انتخاب کنید</option>
  <option value="iran">ایران</option>
  <option value="usa">آمریکا</option>
  <option value="uk">انگلیس</option>
</select>

<!-- چک‌باکس -->

<!-- رادیو باتن -->
<p>جنسیت:</p>
<label><input type="radio" name="gender" value="male"> مرد</label>
<label><input type="radio" name="gender" value="female"> زن</label>

<!-- ورودی فایل -->
<label for="file-upload">انتخاب فایل:</label>
<input type="file" id="file-upload" name="file-upload" accept=".jpg,.png,.pdf">

<!-- ورودی رنگ -->
<label for="favcolor">رنگ مورد علاقه:</label>
<input type="color" id="favcolor" name="favcolor" value="#4f46e5">

<!-- ورودی محدوده -->

۱۴. جاسازی محتوا

با استفاده از iframe و object می‌توانید محتوای خارجی را در صفحه خود جاسازی کنید.

۱۵. اعتبارسنجی و بررسی HTML

برای اطمینان از درست بودن کد HTML خود، می‌توانید از ابزارهای اعتبارسنجی استفاده کنید.

ویژگی‌های اعتبارسنجی در فرم‌ها

<!-- اعتبارسنجی پایه -->
<form action="/submit" method="POST">
  <label for="email">ایمیل (الزامی):</label>
  <input type="email" id="email" name="email" required>

  <label for="age">سن (بین ۱۸ تا ۶۵):</label>
  <input type="number" id="age" name="age" min="18" max="65" required>

  <label for="phone">تلفن (الگوی خاص):</label>
  <input type="tel" id="phone" name="phone" pattern="[0-9]{4}-[0-9]{3}-[0-9]{4}">

  <label for="message">پیام (حداقل ۱۰ کاراکتر):</label>
  <textarea id="message" name="message" minlength="10" maxlength="500" required></textarea>

  <button type="submit">ارسال</button>
</form>

نسخه‌های مختلف HTML

  • HTML5: آخرین نسخه استاندارد فعلی (DOCTYPE: <!DOCTYPE html>)
  • HTML 4.01: نسخه قدیمی با Strict, Transitional, Frameset
  • XHTML: HTML به صورت XML (نیاز به تگ‌های بسته و حروف کوچک)

۱۳. تگ‌های قدیمی و منسوخ (CSS استفاده کنید)

<!-- برای چشمک زدن -->

<!-- برای خط زیر -->

<!-- لیست‌های قدیمی -->
<menu>منو قدیمی</menu> ← <ul> استفاده کنید
<dir>دایرکتوری</dir> ← <ul> استفاده کنید

<!-- فریم‌ها (قدیمی) -->

تمام تگ‌های HTML5 پوشش داده شدند

این لیست پیشرفته‌ترین و کامل‌ترین راهنمای تگ‌های HTML5 است. همه تگ‌های استاندارد، ویژگی‌ها و مثال‌های عملی پوشش داده شده‌اند.

این آموزش یک نقطه شروع عالی برای ورود به دنیای طراحی وب است.

قدم بعدی؟ یادگیری CSS برای استایل‌دهی و JavaScript برای افزودن تعامل.
