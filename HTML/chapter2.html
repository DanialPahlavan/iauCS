Practice Exercise

Create a webpage that demonstrates different text elements:

  1. Add an H1 heading for the page title
  2. Add H2 headings for each section
  3. Create a paragraph explaining your favorite hobby
  4. Add a bulleted list of things you need for that hobby
  5. Add a numbered list of steps to get started
  6. Format some text as bold and italic
  7. Add a horizontal rule to separate sections
